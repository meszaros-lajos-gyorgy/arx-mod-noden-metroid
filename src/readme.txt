------------------------------------------------------
 Noden Metroid mod for Arx Fatalis (work in progress)
------------------------------------------------------

Yet another silly effort of turning Arx Fatalis into a Metroid game

--------------
 Installation
--------------

Unpack the contents of this zip and add it into your Arx Fatalis/Arx Libertatis game's root folder.

Using Fredlllll's Arx Libertatis mod manager is recommended for easy install and uninstall:
https://github.com/fredlllll/ArxLibertatisModManager

----------
 Features
----------

- replaces the menu background with a custom Super Metroid theme image
- replaces menu music with Super Metroid menu theme
- replaces lava ambience on the dwarven level with Magmoor caverns theme from Metroid Prime 1
